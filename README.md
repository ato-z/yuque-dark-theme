 ## 🌠开发选项 ☀️
 > 运行指令后可继续拓展开发, `index.ts` 为人口文件。 后续有修改或者功能添加往这里添加
 
 ### 🌜popup 选项卡🌛
 > `src/popup/index.ts` 是设置弹窗, 更多配置在此添加. `src/popup/index.html` 为布局。
 ```sh
 # 在浏览器端打开可边预览边开发
 npm run dev:popup
 ```
 
### 🛠️ build 打包 📦
```sh
# dist内的文件每次打包都会重新生成
npm run build
```
